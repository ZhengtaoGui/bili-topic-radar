# Web 看板改版设计稿(给 Codex 实现的依据)

目标:把现在朴素、术语化的看板,改成**美观、专业、有教程、每个字段都看得懂**的样子。
面向**不懂技术的 UP 主**。所有文案用下面给定的中文,别再暴露 `top_comments`/`age_days`/
`near_90d_ratio` 这类原始字段名给用户看。

Claude(本稿)负责文案 + 信息架构;Codex 负责视觉实现、布局、交互、tooltip、帮助面板。
欢迎 Codex 在保持下列文案/信息不丢的前提下,提出更好的视觉与交互方案。

## 3. 输入表单(每个字段:人话标签 + 灰色帮助文字 + ⓘ tooltip)

**字段1 — 选题关键词** `*必填*`
- 标签:**选题关键词**
- 帮助文字:你想做的视频主题
- placeholder:`例如:MCP、Claude Code、机械键盘、露营`
- ⓘ tooltip:雷达会拿这个词去 B站 搜同主题视频。用**观众真正会搜的词**最准。

**字段2 — 额外搜索词** `(可选,每行一个)`
- 标签:**额外搜索词(可选)**
- 帮助文字:观众可能用的其他搜法,帮你搜得更全;留空也行
- placeholder:`MCP 教程`(换行)`MCP 实战`(换行)`MCP 原理`
- ⓘ tooltip:同一个题,观众搜法不同(教程/实战/入门/原理…)。多给几个词,覆盖更全、角度地图更准。
- 加一个小按钮:**「+ 自动加常见后缀」** —— 纯前端,把当前主关键词拼上
  `教程 / 实战 / 入门 / 原理 / 配置` 填进文本框(方便不知道填啥的人一键补全)。

**字段3 — 每个热门视频抓几条评论**
- 标签:**抓多少热评**
- 帮助文字:对播放最高的前几个视频抓热评(挖"观众想要什么"),默认 10
- 默认值 10,范围 0–50
- ⓘ tooltip:评论区是金矿——观众喊的需求就是你视频的差异化方向。数字越大越慢。

**主按钮:** **🚀 开始扫描**(替换"采集证据")
**按钮下方小字:** 扫描会真去 B站 搜,约 30–60 秒(限速对 B站 友好),请耐心等。

## 4. 统计卡(5 张)—— 人话名 + 数值 + 一句解读徽章

| 改后名称 | 字段 | 解读徽章(按值变色) | ⓘ tooltip |
|---------|------|--------------------|-----------|
| **相关视频数** | total_unique_videos | — | 全站找到多少个同主题视频。 |
| **近90天占比** | near_90d_ratio | ≥50% 绿「🔥上升期」/ 30–50% 灰「持续产出」/ <30% 红「📉降温·成熟」 | 近三个月新视频占比。越高=题在升温、新人也有机会;越低=热度过了。 |
| **头部垄断度** | top3_play_share | ≥45% 红「头部锁死·难挤」/ 35–45% 灰「中等」/ <35% 绿「流量分散·好进」 | 播放最高的 3 个视频吃掉了多少总播放。越高=被大 UP 垄断、新人越难出头。 |
| **最高播放** | max_play | — | 这个题里跑得最好的视频播放量。 |
| **中位播放** | median_play | — | 一半视频低于这个数。和"最高播放"差距越大=越长尾(多数视频没量)。 |

> 数值格式:播放用千分位;占比用百分比一位小数。

## 6. 使用教程(帮助面板 / 弹窗,点顶部按钮打开)

标题:**怎么用这个雷达?**

**这是什么**
做视频前的"选题尽调"。输入一个选题,它扫遍 B站 同主题视频,告诉你:竞争多不多、
哪个角度还空着、观众在评论区求什么。

**三步用法**
1. **输入选题** —— 填一个关键词(再可选填几个额外搜法),点「开始扫描」。
   懒得想?直接点上面的「看示例」秒看效果。
2. **读结果** —— 顶部 5 个数字是"竞争态势";下面表格是所有同主题视频
   (可按播放/日期排序);点 ➕ 展开看每个视频的高赞评论(观众的真实需求)。
3. **拿去出报告** —— 把这些"证据"交给 Claude(bilibili-topic-radar skill),
   它会做角度聚类、挖评论缺口,给你 **3 个差异化选题方案 + 做/不做建议**。

**怎么快速读那几个数**
- 近90天占比**高** → 这个题正在升温,值得做。
- 头部垄断度**高** → 流量被几个大 UP 吃光,新人难出头,慎入。
- 中位播放远低于最高播放 → 长尾严重,多数视频其实没什么量。

**重要**
本页只展示"原料证据",**不做 AI 判断**;完整选题报告(角度/缺口/三方案)由 Claude
产出 —— 纯前端做不了 LLM 判断,这是刻意的。

## 7. 视觉方向

- 专业、干净、有呼吸感。可沿用现有青绿(teal)主色,或升级到更精致的配色,Codex 定。
